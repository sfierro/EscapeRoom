# Escape Room

A Svelte 5 project featuring an escape room experience.

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173` and should open automatically in your browser.

### Building

To create a production build:

```bash
npm run build
```

### Preview Production Build

To preview the production build:

```bash
npm run preview
```

## Project Structure

- `src/App.svelte` - Main application component with the home screen
- `Assets/HouseBackgrounds/` - Background images for the escape room
- `index.html` - Entry point HTML file
